# React Router v7 Migration Summary

## Changes Overview
- **Dependencies**: Upgraded to `react-router` v7, `react` v19, and `tailwindcss` v4.
- **Architecture**: Moved from a standard Vite SPA to React Router "Framework Mode".
- **SSG**: Configured static site generation via `react-router.config.ts` using a dynamic crawler for project routes.
- **Routing**: Replaced `HashRouter` with `BrowserRouter` (implied by Framework Mode) and defined routes in `src/routes.ts`.
- **Data Loading**: Migrated `useEffect` data fetching to RRv7 `loader` functions.
- **Components**: Updated `Home` and `ProjectDetail` to use `useLoaderData`.
- **Layout**: Established `src/root.tsx` as the document root and fixed layout regressions.

## Build Output
- The build now generates a full static site in the `dist/` directory (previously `build/client`).
- SSG produces directory-based `index.html` files (e.g., `dist/minimal-blog/index.html`) for clean URLs.

## Verification
- Verified build success and SSG HTML content.
- Visual verification performed via Playwright screenshots.
