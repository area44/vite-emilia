This is a comprehensive test project designed to showcase the new masonry layout implementation.

The layout uses CSS columns to create a fluid, masonry-style gallery. Landscape images are configured to span across both columns to maintain their visual impact and reset the masonry flow where necessary.

Portrait and square images will sit side-by-side, filling the vertical space efficiently. This project includes a variety of image orientations to ensure the layout remains robust and visually appealing on all screen sizes.

### Testing features:

- Responsive column counts (1 on mobile, 2 on tablet/desktop)
- Landscape image spanning (`column-span: all`)
- Tightened gaps for a modern aesthetic
- Constrained container width for large screens
- Proper vertical spacing between images
